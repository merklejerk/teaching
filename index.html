Core Components

Sigmoid Function (σ): The activation function used by the neuron. It squashes any input value to a range between 0 and 1.

σ(z) = 1 / (1 + e-z)

Chain Rule Application

We use the chain rule to find the partial derivatives of the loss L with respect to each parameter (w, b). The general form is:

∂L/∂w = (∂L/∂y) * (∂y/∂z) * (∂z/∂w)

Where:

  • z = w*x + b (the linear combination)
  • y = σ(z) (the sigmoid activation)
  • L = (y_true - y)² (the loss function)

Step 1: Derivative of Loss w.r.t. Output (∂L/∂y)

∂L/∂y = 2 * (ytrue - y) * (-1) = -2(ytrue - y)

Step 2: Derivative of Activation w.r.t. Linear Input (∂y/∂z)

The derivative of the sigmoid function σ(z) is σ(z) * (1 - σ(z)).

∂y/∂z = y * (1 - y)

Step 3: Derivative of Linear Input w.r.t. Parameters

∂z/∂w = x

∂z/∂b = 1

Combining the parts:

For w:

∂L/∂w = [-2(ytrue - y)] * [y(1-y)] * [x]

For b:

∂L/∂b = [-2(ytrue - y)] * [y(1-y)] * [1]
